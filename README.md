# 🎓 Student Placement Management System

A web-based application to manage and track student placement activities including job postings, applications, and results.

---

## 📌 About the Project

Managing placement activities manually is complex for colleges. This system provides a centralized platform for students, placement officers, and companies to interact efficiently.

Built as part of B.Tech CSE (AI/ML) coursework.

---

## ✨ Features

- 👩‍🎓 Student registration and profile management
- 🏢 Company job posting management
- 📋 Application tracking system
- ✅ Placement results and reporting
- 🌐 Easy-to-use web interface

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Language | Python |
| Backend | Flask |
| Frontend | HTML, CSS, JavaScript |
| Database | SQL / MySQL |

---

## ⚙️ How to Run

```bash
# 1. Clone the repository
git clone https://github.com/ashwitha89/student_placement_management_system.git

# 2. Go to the project folder
cd student_placement_management_system

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the app
python app.py
```

Then open your browser and go to: `http://localhost:5000`
